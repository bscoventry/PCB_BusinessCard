# Printed Circuit Board Business Card
![BusinessFront](Fabbed/PXL_20260406_171045116.png)
![BusinessFront](Fabbed/PXL_20260406_171055896.png)
Repo for the near-field coupled business card design. Repository contains PCB gerber files as well as Bill of materials and instructions

JLCPCB Options: Get the thinnest board possible, I recommend 0.8 or lower. The board file is designed such that everything arrives as a business card with no extra cuts needed.
